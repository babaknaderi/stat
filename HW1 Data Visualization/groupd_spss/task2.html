Example

Provide a script and html file which generates/shows a boxplot separated for each game, visualizing the video quality ratings (VQ) for the condition IDs15, 43, and 66 (they are all the same) representing the following combination of parameters: resolution: 480, framerate: 30, bitrate: 300 (number of plots: 6,dataset: Gaming_Video_Quality_dataset.xlsx)

Setup and transfering data format

Click Select Cases > Select “If condition is satisfied” > click “If…” > Define condition “Condition = 15 OR Condition = 43 OR Condition = 66”

Click "Split File" > Split File-Modal: Check option “Organize output by groups” and add “Game” to be the variable that the groups are based on

Performing the test

  1. Click Graphs > Chart Builder > Choose “Clustered Boxplot”-Plot type
  2. Define Graph:
    • Add “VQ” variable on the y-axis
    • Add “Condition_params” variable on the x-axis
    • Add the “Game” variable to the “Cluster on X”-section
    • Give the x- and y-axis meaningful names
