Example

Provide a script and html files which generates/shows a stacked histogram (sum of both groups on top of each other for each bar/bin) visualizing the age distribution (grouped by gender) of participants in the speaker likeability dataset (number of plots: 1, dataset: speaker likeability/speech_quality_ratings.xlsx)

Setup and transfering data format

Simply import the data into SPSS, no addiontal steps are needed.

Performing the test

  1. Click Graphs > Chart Builder > Choose “Stacked Bar”-Plot type
  2. Define Graph:
    • Add “Age” variable on the x-axis
    • Add the “Gender” variable to the “Stack: set colour”-section
    • Give the x- and y-axis meaningful names
