Example

Put the age values in bins of: 18-25, 26-35, 36-50, 50+.

Dataset: Speech_quality_crowdsourcing_dataset.xlsx

Setup and transfering data format

Import data from the “CS participants” tab of the excel file and make sure to check the option “Read variables names from first row of data”.

Performing the test

  1. Transform -> Compute Variable
  2. Give the new variable the label “Age”
  3. In the “Numeric Expression” field write the expression for calculating the participants age (current year minus “Birthyear” variable)
  1. Transform -> Recode into Different Variables
  2. Add the “Age” variable to the list of variables to be recoded and enter in the “Name” field the name of the new variable (e.g. “Age_Category”)
  3. Click on “Old and New Values” to define the new categories -> In the “Old Value” section choose the values you want to bind in a specific category and in the “New Value” section choose the value you want to recode to.

Note that each of the for categories gets a number from 1 to 4 assigned, so that for e.g. the values from 18 to 25 will be replace by 1, the values from 26 to 35 by 2 and so on

  1. Navigate to the “Variable View” -> Click in the “Value” column of the “Age_Category” variable
  2. Create a label for each of the values (1,2,3,4)

Make sure that the value of the “Decimals” column for the “Age_Category” value is null
