Setup and transfering data format

Import all VQ data from Excel file

Performing the test

  1. Click on Select cases
  2. Check “If condition is satisfied” in the “Select” section click on “If…”-button ”
  3. Define condition “Condition = 26 AND Game = "Game4"

  1. Click on Analyze -> Descriptive Statstics -> Explore… ->
  2. Add the “VQ” variable in the “Dependent List” -> Make sure the “Both”-option is checked in the “Display” section
  3. -> Click on “Plots” -> Check the “Normality plots with tests” option

  1. Click on Analyze -> Descriptive Statstics -> Q-Q Plots… ->
  2. Add the VQ variable to the list of “Variables” and select in the “Test Distribution” section “Normal”
