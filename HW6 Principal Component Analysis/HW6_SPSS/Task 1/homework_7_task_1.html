Example

Use all collected ratings to derive possible dimension and item loadings of interpersonal speaker characteristics. Note that also an inverse interpretation of the ratings is possible. Please split the data in male and female speakers before performing the factor analysis.
File to use: likeability_dimension_ratings.csv

Setup and transfering data format

Performing the test

Analyze > Dimension Reduction > Factor

In the “Factor Analysis” modal, add the speaker characteristics to the “Variables” list.

1. Click on “Descriptives…” button in the "Factor Analysis" modal and check in the “Correlation Matrix” section the “Coefficients” and the “KMO and Bartlett’s test of sphericity” options.

2. Click on “Extraction…” and in the “Display” section, check the “Scree plot” option. Furthermore, the extraction of dimensions follows the “Principal components” method based on Eigenvalues greater than 1.

3. Click on “Rotation…” and choose the “Varimax” method.

4. Click on “Options…”. For a better overview of the loadings of each variable on the corresponding factor, we are only going to visualize the variables that have a loading higher than .40 on the factors.

Output interpretation

In both female and male, the KMO (Kaiser-Meyer-Olkin Measure of Sampling Adequacy) value exceeded .5 with .932 and .921 respectively. Thus, we can conduct a factor analysis on the dataset.

In the “Total Variance Explained”, each of the 34 speaker characteristics has a quality scored called “Eigenvalue” calculated. Only components with a high Eigenvalue are likely to represent a real underlying factor. The number of underlying factors correspond to the number of components with an Eigenvalue of at least 1. For both gender types, the 34 variables seem to measure 7 factors.

The scree plots visualize the Eigenvalues of the characteristics. We can observe, that for both female and male seven of them have values bigger than 1.

The rotated component matrix shows which variables which factor measure. According to literature, for a sample size of minimum 100 participants, loadings of .30 or higher can be considered significant. Consequently, only variables with a loading higher that .40 are illustrated in the matrix.

Validation

In the "Rotated Component Matrix" it can be seen that variables with a negative meaning (e.g. gleichgueltig, gelangweilt) have a negative loading factor. If we compute the Cronbach's alpha of the seven factors, we will get an invalid value. To comput the internal reliability we need to rescale the negative variables to the other direction.

For the variables unsympatisch, verstaendnislos, distanziert, gelangweilt, charakterlos, dominant, alt, haesslisch, gleichgueltig, zynisch, aufgesetzt we flip the values by substracting the actual values forma 200.

Transform > Compute Variable... > In the "Compute variable" modal we enter the "flipping" expression (100 - variable_name)

For each the enumerated variables we repeate the calculation process.

For each resulting factor/component we compute the Cronbach's Alpha value separated for the female and male speakers.

Analyse > Scale > Reliability Analysis... > In the "Reliability Analysis" modal, we add the variables of the component for which we are computing the alpha-value and under "Statistics", in the "Inter-Item"- section, we check the "Correlations" option.

We repeate the calculation process for each component for both female and male speakers.

The first factor has an excellent internal consistency (Cronbach’s alpha = .914). With values between 0.7 and 0.8, the second and forth component have an acceptable consistency. Problematic are the fifth and seventh factors, which have a Cronbach’s alpha value lower than 0.5.
