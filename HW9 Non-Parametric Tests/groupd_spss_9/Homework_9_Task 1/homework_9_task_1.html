Task 1

1.RQ: Does changing the bitrate (independent variable: 2000, 4000, 6000, 50000 kbps) have a significant effect on the video quality (VQ) ratings (dependent variable). Please consider all ratings at a resolution of 1080p and framerate of 60 fps for the FIRST game. Use the ratings provided in the gaming video quality dataset.

Setup and transfering data format

Simply load the data set into SPSS. Check “Read Variable Names from the First Row of Data”

Performing the test

We have to filter the data according to the given conditions which are resolution of 1080p and framerate of 60 fps for the FIRST game. Data > select cases. In the cases set the conditions shown. This will filter all the unselected conditions.

Now we have to restructure the data, making seperate columns for each bitrate. "Data">"Restructure". Choose "Restructure selected cases into variables" and click "next". Then add "Bitrate" to index variable and "PID" to identifier variable. and finish the wizard without changing anything else.

Now, we can do friedman test. Analyse>Nonparametric tests>lagacy dialogs>K related samples. Add VQ.2000, VQ.4000, VQ.6000 and, VQ.50000 variables to "Test Variables". Check "Friedman" box. Click on "Exact" and choose "Exact". Click "continue" and the "Statistics" menue check "Quartiles"

Now that we know that overall changing bitrate does have an effect on VQ. We would like to know which indivisual bitrate changes effect the VQ. For that, we are going to do a Wilcoxon signed rank test. Analyse>Nonparametric tests>lagacy dialogs>2 related samples. Add all possible pairs of the VQ at 4 bitrates.

Output

There is a statistically significant difference in VQ depending on the Bitrate, χ2(3) = 56.716, p < 0.001. Post hoc analysis with Wilcoxon signed-rank tests was conducted with a Bonferroni correction applied, resulting in a significance level set at p < 0.01. Median (IQR) VQ for bitrate 2000, 4000, 6000 and 50000 were 2.90 (2.45 to 3.40), 3.70 (3.00 to 4.50), 4.40 (3.70 to 4.90) and, 5.30 (5.00 to 5.90), respectively. There were significant differences between VQ ratings at every bitrate as shown.
