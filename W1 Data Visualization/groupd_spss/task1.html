Example

Provide a script and html file which generates/shows for each of the 3 resolutions a separated chart, visualizing the mean values of the video quality ratings and the 95% confidence interval in form of a bar plot. Use all available framerate-bitrate pairs on the x-axis and the game as grouping variable (number of plots: 3, dataset: Gaming_Video_Quality_dataset.xlsx)

Setup and transfering data format

Import the dataset in SPSS. Optional: The type of the "resolution" row is recognized as numeric data. It can be changed from "numeric" to "string", because it is actually nominal data. Note that this will not affect the output for this task.

Performing the test

Split File > Check option “Organize output by groups” and add “Resolution” to be the variable groups are based on

  1. Click Graphs > Chart Builder > Choose “Clustered Bar”-Plot
  2. Define graph
    • Add “VQ” variable on the y-axis
    • Add “Condition_params” variable on the x-axis
    • Add the “Game” variable to the “Cluster on X”-section
    • Give the x- and y-axis meaningful names
  3. Show the 95% CI: Click on the chart bars > Check option “Display error bars” > In the “Error Bars Represent”-section check the “Confidence intervals”-option and give it a “Level”-value of 95
