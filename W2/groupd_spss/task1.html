Example

Calculate (mean, std, 95%CI) per condition.

Dataset: Speech_quality_crowdsourcing_dataset.xlsx

Setup and transfering data format

  1. In the Excel-file compute the values of the “condition” column based on the “files” column. You can use the MID()/TEIL() function which returns the substring of the filename corresponding to the condition value, e.g. MID(A2;7;1)
  2. Open the “CS-per file” worksheet in SPSS and make sure to check the option “Read variables names from first row of data”.

In the "Data View" tab, you can see that the ratings are spread over multiple columns. In order to calculate the descriptive statistics, you need to create a new variable that holds all ratings from P1-P18 in one column. To achieve that, perform the following steps:

  1. Data -> Restructure -> Select “Restructure selected variables into cases” -> Select “One” for the number of variable groups
  2. Add all the ratings P1 to P18 in the “Variables to be Transposed” section and give the new variable a significant name (e.g. “reviews”)
  3. Add the “condition” and “files” variables to the “Fixed Variable(s)” section in order to keep them when restructuring the data
  4. Select “One” for the nr. of index variables
  5. In the next step select “Variable names” for the index values and rename the index (e.g. “ratings_no”)
  6. In the “Handling of Variables not Selected” section check the “Drop variable(s) from the new data file” as we already specified that we want to keep the “files” and “condition” columns

You want to split the data regarding the “condition” variable in order to create an output per condition.

  1. Split File > Check option “Organize output by groups”
  2. Add “condition” to be the variable groups are based on

Performing the test

To perform the actual task, follow these steps:

  1. Analyse -> Compare Means -> One-Sample T Test
  2. Add the “ratings” variable to the list of “Test Variable(s)”
  3. Click on the “Options…” button -> Enter 95 for the value of the “Confidence Interval Percentage”
