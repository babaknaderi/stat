Example

Calculate the suitable correlation coefficients to compare the ratings from the crowd (CS-per condition) with those from the lab (Lab-per condition). Show the data distribution in a scatter plot.

Dataset: Speech_quality_crowdsourcing_dataset.xlsx

Setup and transfering data format

Data > Merge Files > Add Variables (we want to add the MOS variable from “Lab-per condtion” to “CS-per condition”)

In the “Merge Method” tab, select “One-to-one merge based on key values” and the key variable should be the “condition” variable, as this one is the one column both files have in common.

In the “Variables” tab, add “MOS_per_lab” and “MOS_per_condition” to the “Included Variables” section. The key for both variables, should be the unique variable “condition”. All the other datasets variables are excluded from the merging process.

Result of merge: dataset with 3 variables

Performing the test

The variables “MOS_per_condition” and “MOS_per_lab” are both metric variables. To calculate the correlation coefficient between the two of the we chose to compute the Pearson’s Linear Correlation Coefficient, which is suited for continuous variables. Before calculating it, we want to investigate if there are outliers in the data for “MOS_per_condition” and “MOS_per_lab”. We do this by illustrating the two variables into boxplots.

From the boxplots we can notice that, in both cases, the data does not incluede any outliers.

Analyze > Correlate > Bivariate > Add both “MOS_per_condition” and “MOS_per_lab” to the “Variables” list and check in the “Correlation Coefficients” section the option “Pearson”.

Output

We can see that both Pearson Correlation coefficients for the combinations “MOS_per_condition” – “MOS_per_lab” and vice versa are significant at the level 0.01. This suggests that both variables are correlated.

Graphical Visualization

Choose in the “Chart Builder” modal the “Scatter/Dot” plot type and pick the “Simple Scatter with Fit Line” to illustrate the correlation. On the X- and Y-axis the variables can be placed arbitrary. In the e.g. the “MOS_per_lab” variable is illustrated on the x-axis and “MOS_per_condition” on the y-axis.

From the plot, it is notable that a correlation between the variables exists, suggested by the line like distribution of the points.
