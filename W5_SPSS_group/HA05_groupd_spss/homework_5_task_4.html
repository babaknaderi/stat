Example

Use multiple linear regression to predict video quality (VQ) based on fragmentation (VF), unclearness (VU) and discontinuity (VD).

Setup and transfering data format

Choose analyse → Regression → Curve Estimation…

Add VF to “Dependent(s)” and Bit_per_pixel to “Independent” → In “Models” section check “Exponential”

Choose Analyse → Regression → Curve Estimation… → Add VU to “Dependent(s)”and Bit_per_pixel to “Independent” → In “Models” section check “Cubic”

Choose Analyse → Regression → Curve Estimation… → Add VF to “Dependent(s)”and Framerate to “Independent” → In “Models” section check “Exponential”

Output

Output for non-linear regression (exponential fit) for VF based on bitperpixel
Output for non-linear regression (polynomial fit 3rd order) for VU based on bitperpixel
Output for non-linear regression for VD based on framerate
