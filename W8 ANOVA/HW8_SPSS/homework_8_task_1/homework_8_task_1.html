Task 1

1.RQ: Does increasing the bitrate or changing the game (independent variables) have a significant effect on the video quality (VQ) ratings (dependent variable). Please consider only ratings at a resolution of 1080p and a framerate of 60 fps (conditions 36 and 50). Use the ratings provided in the gaming video quality dataset.

Setup and transfering data format

Simply load the data set into SPSS. Check “Read Variable Names from the First Row of Data”

Performing the test

We have to filter the data according to the given conditions which are 50 and 36. Data > select cases. In the cases set the conditions shown. This will filter all the unselected conditions.

Now we have to compute the variables on the bases of games. We want game variable to be numerical for this purpose. Transform > Recode into different variables. Make to a new variable “game_num” according to the conditions defined in the picture. “game_num” will be used to compute variables

Now we have to compute the 4 variables having VQ based on bitrate and game, We will have VQ_1_2000 representing the VQ for game1 and bitrate 2000 and so on.

Now that we have the required data, we will now try to find the outliers in VQ variable. As shown in the plot, there are no outliers. Analyze > Descriptive Statistics > Explore… > Add “VQ” to “Dependent List” > Click on “Statistics…” button > In the “Statistics” modal, check the option “Outliers” > Click on “Plots…” button > under the “Descriptive”-section uncheck “Stem-and-leaf”

Analyze > Descriptive Statistics > Explore… > Add all 4 computed variables to “Dependent List” > Click on “Plots…” button > under "Plots" section, check "Normality plots with tests" and under the “Descriptive”-section uncheck “Stem-and-leaf”. As shown, two of the variables arent normally distributed. (VQ_1_2000 and VQ_6_2000)

Corecting the data for normality

To make all the variable "normal", we look at the boxplots of the 4 variables we computed earlier. The boxplots of "VQ_1_2000" and "VQ_6_2000" show some extreme outliars. We remove the outliars by limiting the values of "VQ_1_2000" as (VQ_1_2000>=2)&(VQ_1_2000<=3.6) and "VQ_6_2000" as (VQ_6_2000>2.6)&(VQ_6_2000<=3.4), shown below. The limits are determind by turning on the grid lines and scaling on boxplots. The resultant boxplots dont show any significant outliars:

Checking the normality of the 4 variables again

As shown below, all 4 variables now conform to the conditions of normality:

Doing the two-way repeated ANOVA test

since we have two input variables (bitrate and game) and the experiment was repeated with the same participants. Hence, we do two-way repeated ANOVA. Analyse > General Linear Models > Repeated Measures. Add "Game_numf" and "Bit_rate" with 2 levels as factors. Map these factors with 2 levels on 4 computed variables (that is "VQ_1_2000" and so on...). Click on Estimated Maeginal Means and add "Game_numf" "bit_rate" and "Game_numf*bit_rate" to "display means for". Finally, in the option menu, check "descriptive statistics" and "Estimate of effect size"

Output

As there is no signicance on sphericity test, hence we check the "Within-Subjects Effects" table

as shown, there is no interation effect and no effect of game on the Video quality. But, bitrate has large effect. F(1,13)=127.806, p<0.001,eta square= 0.908. The complete output is contained in the "within_subject.pdf"

nothing can be said about the signicance of means since there is no pairwise comparison table generated.
