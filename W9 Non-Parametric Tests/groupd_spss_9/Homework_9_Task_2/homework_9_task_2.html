Example

Does the testStimulus have a significant influence on speech quality ratings and if yes, for which cases. Please assume that each of the six files was assessed by a different set of test participants and only use the rating of the first repetition. Use the quality ratings provided in the data set speech_quality_repetition_dataset.

Setup and transfering data format

Data > Select Cases...

The sample size for each group is 37.

Test Requirements

Analyze > Descriptive Statistics > Explore… > Add “rating” to the "Depemdemt List" and “testStimulus” to the “Factor List”

Click on “Statistics…” button > In the “Statistics” modal, check the option “Descriptives”.

Click on “Plots…” button > In the “Plots” modal check the option “Normality plots with tests” and under the “Descriptive”-section uncheck “Stem-and-leaf”.

The Shapiro-Wilk Test showed a significant departure from normality for all groups.

Kruskal-Wallis Non-Parametric Test

Conduct the test.

Analyze > Nonparametric Tests > Independent Samples...

In the Nonparametric Tests modal, check the option "Customize analysis> and navigate to the Fields tab.

In the Fields tab, add rating to the "Test Fields" and testStimulus to "Groups". Change to Settings tab next.

In the Settings tab, check "Customize tests" and check the Kruskal-Wallis 1-way ANOVA option. Under "Multiple comparisons" choose All pairwise. Then click on Run

Output interpretation





Effect sizes are calculated to assess the significance of a result. Since the post hoc test provides information about differences between each group, what we want to calculate is the effect size of the Dunn-Bonferroni test. To calculate Pearson’s Correlation Coefficient r the sample size n and the z-value of each comparison are used:

The effect size of the diffrence between the first and third test stimulus is:

r = |-6,047/ sqrt(74)| = |-6,047 / 8,60| = 0,7


The Kruskal-Wallis-Test confirms that the test stimulus has a significant influence on the speech quality ratings (Chi-Quadrat(5) = 130.897, p < .001). Post hoc comparision (Dunn-Bonferroni tests) indicated that the first testStimulus (M=2.62, SD=.98) and the third testStimulus (M=4.51, SD=.559) differ significantlly regarding the speech quality ratings (z = -6,047, p < .001, r = 0,7) .
